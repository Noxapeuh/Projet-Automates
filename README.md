# Projet-Automates

**Lecture d'automates**

Notre programme permet de lire un automate depuis un fichier .txt, sous le format :
```
états : 0, 1, 2
initial : 0, 1
finaux : 1, 2
transitions :
0, a, 1
0, b, 2
2, a, 1
```

On peut ensuite afficher sa table des transitions, avec des informations utiles comme les états initiaux ou les états finaux.
